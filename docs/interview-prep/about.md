# About Interview Prep

This is just my attempt to keep a consistent rhythm in technical interview prep. It'll have a mix of math puzzles, and daily topics. By no means am I saying this is the best way to prep, it's just the way I've decided to prepare and keep my skills sharp.  


Each week has two connected tracks:

- daily 7-in-30 quant practice, where I attempt 7 quant-style problems in 30 minutes and record the score
- weekday topic work across statistics, SQL, model implementation, ML system design, and applied data science cases

The weekly overview is the main hub. It links to each weekday topic post and each day's 7-in-30 problem page as those pages are created and filled in.

## Weekly rhythm

| Day | Topic block | Focus |
|---|---|---|
| Monday | Statistics | statistics, probability, experimentation |
| Tuesday | SQL | SQL, analytics, data manipulation |
| Wednesday | Model Implementation | models and algorithms from scratch |
| Thursday | ML System Design | ML system design and production architecture |
| Friday | Applied Case Study | applied DS cases and weekly review |

## Quant baseline

| Target | Value |
|---|---:|
| Problems per day | 7 |
| Time limit | 30 min |
| Weekly total | 49 |
